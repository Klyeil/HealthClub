스미스 머신 Smith Machine

사용 방법

  1. 바벨의 높이를 어깨 높이보다 약간 낮게 설정합니다.
  2. 바벨 아래 누워 어깨 너비보다 넓게 바를 잡습니다.
  3. 바를 들어 올린 뒤 천천히 가슴 중앙 쪽으로 내립니다.
  4. 가슴 근육을 수축하며 바를 다시 밀어 올립니다.

랫 풀 다운 Lat Pull Down

사용 방법

  1. 패드에 무릎을 고정하고 바를 어깨 너비보다 넓게 잡습니다.
  2. 가슴을 펴고 바를 쇄골 쪽으로 당겨줍니다.
  3. 광배근의 수축을 느끼며 팔꿈치를 옆구리로 붙입니다.
  4. 천천히 저항을 버티며 팔을 펴줍니다.

레그 프레스 Leg Press

사용 방법

  1. 등받이에 엉덩이와 등을 밀착시킵니다.
  2. 발판에 발을 어깨 너비로 벌리고 위치시킵니다.
  3. 안전바를 풀고 무릎이 90도가 될 때까지 천천히 내립니다.
  4. 발뒤꿈치로 민다는 느낌으로 발판을 밀어냅니다.
